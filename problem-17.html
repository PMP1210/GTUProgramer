Problem 17

Problem statement:-


Write a C program to find factorial of a given number.

Code:-


#include stdio.h
int main()
{
printf("\n________________________________________");
printf("\n\n\tGIVE THE FACTORIAL");
printf("\n________________________________________");

int fac=1,a,i;
printf("\nEnter the number :");
scanf("%d",&a);

for(i=1;i<=a;i++)
{
fac=fac*i;
}
printf("\nFactorial is=%d",fac);
return 0;
}
