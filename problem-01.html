Problem 01

Problem statement:-


Write a program to perform as calculator. ( Addition , Subtraction , Multiplication , Division)

Code:-


#include <stdio.h>

void main()
{
printf("\t\tCalculator");
printf("\n=====================================================");

float a, b;

printf("\n\nEnter A : ");
scanf("%f", &a);

printf("\nEnter B : ");
scanf("%f", &b);

printf("\n_____________________________________________________");

printf("\n\nAddition of A (%.2f) And B (%.2f) is : %.2f", a, b, a + b);

printf("\n\nSubtraction of A (%.2f) And B (%.2f) is : %.2f", a, b, a - b);

printf("\n\nMultiplication of A (%.2f) And B (%.2f) is : %.2f", a, b, a * b);

printf("\n\nDivision of A (%.2f) And B (%.2f) is : %.2f", a, b, a / b);

printf("\n_____________________________________________________\n\n");

}
Description:-

This code is a C program that functions as a simple calculator. It performs addition, subtraction, multiplication, and division on two user-inputted numbers, a and b. The program prints out the result of each operation with appropriate formatting and uses the scanf function to receive input from the user. The final output is displayed on the console screen.
