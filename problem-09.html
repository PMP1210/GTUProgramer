Problem 09

Problem statement:-


Write a program to read marks of a student from keyboard whether the student is pass or fail. (using if else)

Code:-


#include stdio.h
int main()
{
printf("\n________________________________________");
printf("\n\tFIND THAT STUDENT IS PASS OR FAIL");
printf("\n________________________________________");

float a;
printf("\nEnter the mark :");
scanf("%f",&a);

if(a>=33)
{
printf("\nThe student is Pass.");
}
else
{
printf("\nThe student is Fail.");
}

return 0;

}
