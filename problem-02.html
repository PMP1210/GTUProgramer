Problem 02

Problem statement:-


Write a program to find the area of triangle. (a=h*b*0.5) a=area , h=height , b=base

Code:-


#include <stdio.h>
int main()
{
printf("\n_______________________________");
printf("\n\tAREA OF TRIANGLE");
printf("\n_______________________________");
int a,b;
printf("\n\nEnter the hight value:");
scanf("%d",&a);
printf("\n\nEnter the base value:");
scanf("%d",&b);

printf("\n\nArea of triangle is=%.2f",0.5*a*b);
return 0;
}
Description:-

This code is C program that functions as a finding the area of the triangle.First it takes two values from the user in variable a & b using of printf & scanf.Than after both values are multiply with each other and the value of that will multiply with 0.5 .The final output is displayed on the console screen.
